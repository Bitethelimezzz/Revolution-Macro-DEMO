What is Revolution?

Revolution Macro is a new macro for Bee Swarm Simulator, created from the ground up using modern technologies. It's designed with accessibility and modernity in mind— Revolution supports both MacOS and Windows with an innovative web-based interface. Revolution is completely free and always will be, with a mission to address critical performance and customizability issues in existing macros.

Features

Vic Hop

Collaborative passive/searcher vic-hop that can scale to dozens of accounts working together.

Multi-Instancing

Run multiple accounts in a single desktop session (limited to passives only on Windows).

Account Management

Switch accounts on-the-fly and manage private servers with ease.

Intelligent Pathing

A no-reset pathing system that goes directly from point to point, improving your macro’s efficiency.

Is It Safe?

Yes! Revolution is completely safe to use on both Mac and Windows. Revolution does not interact with the Roblox process; it only sends keyboard and mouse inputs and captures the Roblox window for essential macro functions. It does not share any sensitive data with the internet but does enable anonymized telemetry by default, which can be opted out of at any time.

About the Author

Revolution is a solo project created by Liam, a passionate developer with over 10 years of programming experience and 3 years in full-stack development. Frustrated with existing macro solutions and the toxic culture of gatekeeping in the Bee Swarm Simulator community, Liam started Revolution as a passion project in July 2023. Revolution was designed to be accessible, free, and built with care from a trusted developer.
